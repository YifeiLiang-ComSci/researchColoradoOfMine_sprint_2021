\begin{abstract}
% covid-19의 대유행은 의료체계에 큰 부담을 가하고 있습니다. 의료 자원의 효율적인 배분을 위해, covid-19 환자의 중증도를 예측하려는 연구가 있어왔습니다. Because chest X-ray (CXR) imagings have been collected over time to observe changes in the lungs caused by covid-19, 이러한 longitudinal imagings을 자동 분석하여 환자의 중증도를 예측하는 model은 의료진들의 부담을 덜어줄 수 있습니다. 하지만 촬영된 imagings의 수가 환자마다 다르기 때문에, 이러한 longitudinal data를 static data와 통합하여 중증도를 예측하는 것은 challenging합니다. 이러한 어려움을 극복하기 위해, 우리는 semi-supervised representation learning model to transform the longitudinal imagings into fixed length vector which can be readily integrated with the static data. Armed with the enriched representation from static/dynamic data and their available labels, one can fully utilize the CXR imagings to predict the severity of COVID-19 patients.
